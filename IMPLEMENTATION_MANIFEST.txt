📖 HOW TO USE

FOR EDUCATORS (Minimum Cost):
  1. Install Ollama: curl -fsSL https://ollama.com/install.sh | sh
  2. Pull model: ollama pull llama3.2
  3. Configure: OLLAMA_DEFAULT=true in .env
  4. Start: npm start
  5. Result: Free local AI for simple Q&A, Claude for complex tasks

FOR REGULAR USERS (Balanced Cost):
  1. Set: OLLAMA_DEFAULT=false (default)
  2. Start: npm start
  3. Result: Smart routing automatically chooses best option

FOR TESTING:
  1. Manual: /ollama <message> to test Ollama routing
  2. Claude default: All messages use Claude (Ollama disabled)
  3. Monitor: Check logs for routing decisions
